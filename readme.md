# 列车通信网络攻击检测仿真测试验证平台软件实现

### 本平台建设需求来源于实验室项目《车载网管交换机网络安全防御模块研发》，仓库代码主要用于列车通信网络仿真平台的软件实现。
### 本项目针对列车通信网络特点，计划搭建列车通信网络攻击检测仿真测试验证平台，具备TRDP协议仿真能力和攻击注入能力，便于入侵检测数据获取，展示攻击效果，计划搭建的列车仿真实验平台系统实际结构如下所示。
